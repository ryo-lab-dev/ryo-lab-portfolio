プライバシーポリシー

最終更新日: 2026年2月24日

本アプリは、ユーザーのプライバシー保護を最優先事項として設計されています。

第1条 収集する情報

ScoreVaultは、以下の情報を収集・利用します。

  • ゲームログ（タイトル・評価・レビュー・プレイ時間・画像URL等）
  • GoogleまたはメールによるGoogle Sign-In 等の認証データ（最小限の識別子：メールアドレス・氏名・プロフィール画像URL）
  • Firebase Firestoreによるクラウド同期データ（同期機能を有効にした場合のみ）
  • Steam / IGDB 連携情報（公開ゲーム情報・実績情報）

第2条 利用目的

収集した情報は以下の目的にのみ利用します。

  • アプリ内でのゲームプレイ記録の表示・管理
  • 複数デバイス間でのデータ同期
  • ゲーム情報の自動補完（Steam / IGDB API 連携）

第3条 データ管理

  • 主データはローカルの Isar Database に保存されます。
  • クラウドへの保存は同期機能を有効にした場合のみ行われます。
  • 第三者への個人情報の提供・販売は一切行いません。

第4条 外部サービス

本アプリは以下の外部サービスを利用しています。各サービスのプライバシーポリシーも併せてご確認ください。

  • Firebase（認証・クラウド同期）— Google LLC 提供
  • Steam API（ゲーム情報取得）— Valve Corporation 提供
  • IGDB API（ゲーム情報取得）— Twitch Interactive, Inc. 提供
  • AdMob（広告配信）— Google LLC 提供

第5条 ユーザーの権利

  • アプリ設定からクラウドデータおよびアカウントの削除が可能です。
  • ローカルデータはアプリを削除することで消去されます。
