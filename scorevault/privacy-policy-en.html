Privacy Policy

Last Updated: February 24, 2026

1. Data We Collect

ScoreVault collects the following data to provide its services:

  • Game logs (title, rating, review, playtime, cover image URL, etc.)
  • Authentication data: minimal identifiers (email address, name, profile picture URL) via Google Sign-In
  • Cloud sync: your game log data, encrypted and synced to Firebase Firestore when the sync feature is enabled
  • Steam integration: public game library and achievement data fetched via the Steam API

2. How We Use Your Data

Collected data is used solely for the following purposes:

  • Display and manage your game play records within the app
  • Sync data across multiple devices (when sync is enabled)
  • Auto-complete game information using Steam and IGDB APIs

3. Data Storage

  • Primary data is stored locally in an Isar Database on your device.
  • Cloud storage only occurs when the sync feature is explicitly enabled.
  • Your personal information will not be provided or sold to third parties without your consent.

4. Third-Party Services

This app uses the following third-party services. Please review their respective privacy policies as well.

  • Firebase (Authentication & Cloud Sync) — provided by Google LLC
  • Steam API (Game data) — provided by Valve Corporation
  • IGDB API (Game data) — provided by Twitch Interactive, Inc.
  • AdMob (Advertising) — provided by Google LLC

5. Your Rights

  • You can delete your cloud data or account at any time via the app settings.
  • Local data is automatically removed when the app is uninstalled from your device.
